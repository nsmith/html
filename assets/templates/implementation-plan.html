Risks & Mitigations

RiskSeverityMitigation
Realtime fan-out overwhelms the socket layer on large boards High Batch updates per board and cap subscriptions; load-test at 2× peak.
Optimistic writes diverge from server state on conflict Medium Reconcile on ack; show a retry affordance if the write is rejected.
Comment spam / abuse with no moderation Medium Rate-limit per user and ship soft-delete before public launch.
Migration adds notable index build time Low Create indexes concurrently; run during the low-traffic window.
